## ▶️ How to Use

**Online:** Visit the live URL above — no setup required.

**Locally:**
1. Clone the repo
2. Serve with a local server (do not open index.html directly as a file — audio will not load)
```bash
python -m http.server 8080
```
3. Open `http://localhost:8080` in your browser

**Controls:**
- `→` / `Space` — Next slide
- `←` — Previous slide
- `🔊 Narrating` button — Toggle audio on/off
- `▶ Auto Play` button — Auto-advance through all slides
- Click nav dots to jump to any slide
