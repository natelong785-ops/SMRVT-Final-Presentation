## ▶️ How to Use

1. Open `index.html` in Chrome, Firefox, Edge, or Safari
2. Click **▶ Start Presentation** on the opening screen
3. Navigate using:
   - **→ Arrow key** or **Spacebar** — next slide
   - **← Arrow key** — previous slide
   - **Nav dots** at the top — jump to any slide
   - **Prev / Next buttons** — bottom right corner
4. **🔊 Narrating** button — toggle audio narration on/off
5. **▶ Auto Play** button — auto-advance slides with narration

## 📚 About SMRVT

The **Squadron Manning and Readiness Visualization Tool** is a browser-based, offline-first personnel readiness tracker built for USAF squadron-level use. It provides commanders with real-time visibility into unit readiness through a drag-and-drop manning board, role-based access control, and automatic readiness calculations — with no internet connection required.

**Key technologies:** PouchDB · IndexedDB · Modular ES6 · Drag-and-Drop API · PDF Export
